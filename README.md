# WIKIDLE-REPO
To run the game, run app.py and open the localhost given. (http://127.0.0.1:5000)

Currently the game returns a new Wikipedia article every time the game is reset, as a proof of concept and to make the testing/grading process smoother, but our aim is to have it be a "daily game ('dle'-game)", where there is one article per day, hence the name.
